How does payment verification work?

+

1. You enter your full name and amount

2. Get a reference code to use in transfer

3. Make the bank transfer to our account

4. Submit payment request

5. Admin verifies name matches transfer

6. Funds added upon approval

Why must my name match exactly?

+

This is for security. The admin will:

✓ Check the bank transfer statement

✓ Match it against your submitted name

✓ Prevent fraud and money laundering

✓ Ensure funds come from legitimate source

How long does approval take?

+

Admin approval typically takes 1-24 hours after you submit the request. You'll see the status update in your dashboard.

What if payment is declined?

+

If declined, the admin will show the reason:

• Name mismatch with bank transfer

• Reference code not used

• Transfer amount doesn't match

You can resubmit after fixing the issue.
